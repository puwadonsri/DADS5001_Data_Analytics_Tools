### 1. การกระจายตัวตามแนวรถไฟฟ้า

![Distribution Map](https://user-images.githubusercontent.com/114323892/195903619-2bee77df-a74e-4cd3-87c5-7e5cdffcf8fb.png)

**คำอธิบาย:** แผนที่แสดงตำแหน่งโครงการที่อยู่อาศัย (จุด) เทียบกับแนวรถไฟฟ้า (เส้น) ในพื้นที่กรุงเทพฯ และปริมณฑล โดยใช้ Plotly Mapbox ในการ visualize

**สิ่งที่ค้นพบ:**
- โครงการบ้านเดี่ยวและทาวน์โฮมกระจายตัวออกไปในวงกว้าง ไม่ได้กระจุกอยู่ตามแนวรถไฟฟ้าเท่านั้น
- คอนโดมิเนียมมีแนวโน้มกระจุกตัวตามแนวรถไฟฟ้ามากกว่าอสังหาฯ ประเภทอื่น
- พื้นที่ชานเมืองมีโครงการบ้านจัดสรรจำนวนมากที่อยู่ห่างจากรถไฟฟ้า เนื่องจากราคาที่ดินถูกกว่าและพื้นที่กว้างขวางกว่า

**ข้อควรระวัง:** การ plot ทับซ้อนกันบนแผนที่อาจทำให้มองเห็นความสัมพันธ์ได้ไม่ชัดเจน เนื่องจากโครงการจำนวนมากอยู่นอกเขตกรุงเทพฯ

### 2. ราคาอสังหาริมทรัพย์กับระยะทางจากรถไฟฟ้า

#### 2.1 ราคาเฉลี่ยตามพื้นที่

![Average Price by Area](https://user-images.githubusercontent.com/114323892/195902500-cd3d6288-8e4f-45fe-b25f-b231fa05d574.png)

**คำอธิบาย:** กราฟแท่งแสดงราคาเฉลี่ยของอสังหาริมทรัพย์ในแต่ละจังหวัดในเขตกรุงเทพฯ และปริมณฑล เรียงลำดับจากมากไปน้อย

**สิ่งที่ค้นพบ:**
- **กรุงเทพฯ** มีราคาเฉลี่ยสูงที่สุด เนื่องจากเป็นศูนย์กลางทางเศรษฐกิจ มีรถไฟฟ้าหนาแน่น และมี demand สูง
- **สมุทรปราการ** และ **นนทบุรี** มีราคารองลงมา ซึ่งเป็นจังหวัดที่มีแนวรถไฟฟ้าขยายไปถึง (BTS สายสีเขียว)
- **ปทุมธานี** มีราคาต่ำที่สุดในกลุ่ม — ถึงแม้จะมีรถไฟฟ้าสายสีแดงผ่าน แต่พื้นที่ส่วนใหญ่ยังเป็นชนบทและอยู่ไกลจากใจกลางเมือง

**การวิเคราะห์เพิ่มเติมที่ทำได้:**
- วิเคราะห์ correlation ระหว่างราคากับจำนวนสถานีรถไฟฟ้าในแต่ละจังหวัด
- เปรียบเทียบราคาโครงการที่อยู่ห่างจากสถานี < 500m, < 1km, < 3km เพื่อหา price gradient

#### 2.2 ราคาเฉลี่ยตามแบบบ้าน

![Average Price by House Type](https://user-images.githubusercontent.com/114323892/195902334-7ee897c5-95cc-4ba6-b893-7a5684645399.png)

**คำอธิบาย:** กราฟแท่งเปรียบเทียบราคาเฉลี่ยของอสังหาริมทรัพย์แต่ละประเภท (บ้านเดี่ยว ทาวน์โฮม คอนโด ฯลฯ) โดยแบ่งตามจังหวัด

**สิ่งที่ค้นพบ:**
- **บ้านเดี่ยว (Detached House)** มีราคาเฉลี่ยสูงที่สุดในทุกจังหวัด เนื่องจากมีขนาดที่ดินและพื้นที่ใช้สอยมาก
- **คอนโดมิเนียม (Condo)** มีราคาเฉลี่ยสูงในกรุงเทพฯ แต่ในจังหวัดอื่นมีราคาไม่แตกต่างจากทาวน์โฮมมากนัก — สะท้อนว่าคอนโดในต่างจังหวัดยังไม่เป็นที่นิยมเท่ากับกรุงเทพฯ
- **ทาวน์โฮม (Townhome)** เป็นตัวเลือกที่มีราคาปานกลาง พบมากในพื้นที่ชานเมือง

**การวิเคราะห์เพิ่มเติมที่ทำได้:**
- วิเคราะห์ราคาต่อตารางเมตรของแต่ละประเภท เพื่อเปรียบเทียบความคุ้มค่า
- วิเคราะห์จำนวนห้องนอน/ห้องน้ำกับราคา เพื่อสร้าง price prediction model
- เปรียบเทียบราคาคอนโดที่อยู่ติดสถานี BTS กับที่อยู่ห่างออกไป โดยใช้ distance calculation

## 💡 บทสรุปและข้อจำกัด

### บทสรุป
- การกระจายตัวของอสังหาฯ **ไม่ได้กระจุกตัวตามแนวรถไฟฟ้าทั้งหมด** แต่ขึ้นอยู่กับ **ประเภทของอสังหาริมทรัพย์**
- **คอนโดมิเนียม** มีแนวโน้มกระจุกตัวตามแนวรถไฟฟ้ามากที่สุด
- **บ้านเดี่ยว** มักตั้งอยู่ในพื้นที่ที่ห่างจากรถไฟฟ้า เนื่องจากต้องการพื้นที่ขนาดใหญ่
- ราคาอสังหาฯ ในกรุงเทพฯ สูงกว่าจังหวัดอื่นอย่างมีนัยสำคัญ

### ข้อจำกัด / Challenge
1. ข้อมูลใน dataset ที่ต้องการแมทช์กับเส้นทางรถไฟฟ้า **อาจยังไม่มีความสัมพันธ์กันโดยตรง** — ต้องใช้พิกัด (latitude/longitude) ในการคำนวณระยะทางแทน
2. การ plot โครงการทั้งหมดลงบนแผนทีเดียวทำให้ข้อมูล **ดูทับซ้อนและอ่านยาก** — ควรใช้เทคนิคเช่น heatmap หรือ clustering
3. เนื่องจากข้อมูลมีจำนวนมากและพื้นที่ครอบคลุมทั้งประเทศ การวิเคราะห์เฉพาะเขตกรุงเทพฯ และปริมณฑล **อาจต้องกรองข้อมูลเพิ่มเติม** เพื่อให้ได้ผลลัพธ์ที่แม่นยำขึ้น

## 🚀 แนวทางการวิเคราะห์เพิ่มเติม

### ด้านเทคนิค
| แนวทาง | รายละเอียด |
|--------|-----------|
| **Distance Calculation** | คำนวณระยะทางแบบ Haversine จากพิกัดโครงการ → สถานีรถไฟฟ้าที่ใกล้ที่สุด |
| **Price Gradient Analysis** | วิเคราะห์ราคาเทียบกับระยะทางว่า < 500m, 500m–1km, 1–3km, > 3km แตกต่างกันอย่างไร |
| **Heatmap Visualization** | ใช้ Density Heatmap แสดง clustering ของโครงการ แทน scatter plot |
| **Time Series Analysis** | วิเคราะห์แนวโน้มการพัฒนาโครงการในแต่ละปี เทียบกับการขยายเส้นทางรถไฟฟ้า |
| **Price Prediction Model** | สร้าง Machine Learning model (Linear Regression, Random Forest) เพื่อทำนายราคาจาก distance, property type, facilities |

### ด้านธุรกิจ
| แนวทาง | รายละเอียด |
|--------|-----------|
| **ROI Analysis** | วิเคราะห์ว่านักพัฒนาควรลงทุนในพื้นที่ตามแนวรถไฟฟ้าหรือพื้นที่ชานเมือง |
| **Facility Impact** | วิเคราะห์ว่าราคาโครงการที่มีสระว่ายน้ำ ฟิตเนส รปภ. สูงกว่าโครงการที่ไม่มีเท่าไร |
| **Developer Market Share** | วิเคราะห์ส่วนแบ่งตลาดของผู้พัฒนาแต่ละรายในแต่ละทำเล |
| **Neighborhood Analysis** | วิเคราะห์ความนิยมของแต่ละย่าน (neighborhood) โดยใช้จำนวนโครงการและราคาเฉลี่ย |
